# Hriblet — AI Capsule Memory Extension

**Hriblet** is a Chrome Extension that gives AI chat platforms persistent project memory using a Capsule-based system.

Instead of explaining your project repeatedly to AI assistants, Hriblet allows you to save structured project contexts ("Capsules") and instantly resume work across conversations.

---

## 🌟 Motivation

Modern AI tools are powerful but forget context between chats.

Developers, students, and researchers repeatedly re-explain:

* project goals
* tech stack
* requirements
* architecture decisions

Hriblet solves this problem by introducing **Capsule Memory** — a reusable AI context layer.

---

## 🧠 Core Concept: Capsules

A **Capsule** is a structured memory container containing:

* Project description
* Technology stack
* Features & modules
* APIs used
* Constraints & requirements
* Development notes

Hriblet injects this capsule automatically into AI conversations.

## 📦 Installation (Development Mode)

1. Clone repository

```
git clone https://github.com/YOUR_USERNAME/hriblet.git
```

2. Open Chrome
3. Go to:

```
chrome://extensions
```

4. Enable **Developer Mode**
5. Click **Load Unpacked**
6. Select the project folder

Hriblet is ready ✅

---

## 🚀 Usage

1. Create a Capsule
2. Save project context
3. Open AI chat platform
4. Inject Capsule
5. Continue working instantly
